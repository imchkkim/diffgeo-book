## 제1부: 무대를 세우다

### 1장 — "좌표 하나로는 지구를 담을 수 없다"

**출발 문제:** 지구본을 한 장의 종이에 찢어짐 없이 펼칠 수 있는가? 왜 모든 세계지도에는 왜곡이 있는가?

**패턴:** 전체를 하나의 좌표계로 덮을 수 없지만, 작은 조각마다 좌표를 붙이면 된다. 조각들이 겹치는 곳에서는 좌표 변환 규칙이 있어야 한다.

**정리:** 지구 표면처럼 국소적으로 평평해 보이는 공간은 좌표 조각(chart)들의 모음으로 기술할 수 있다.

**정의:**
- **매니폴드** (국소 평탄 공간 / Locally Flat Space) — 각 점 주변이 ℝⁿ처럼 생긴 공간
- **좌표계/차트** (이름표 체계 / Labeling System) — 매니폴드의 한 조각을 ℝⁿ에 대응시키는 함수
- **아틀라스** (좌표 모음집 / Chart Collection) — 매니폴드 전체를 덮는 차트들의 집합
- **전이함수** (좌표 번역기 / Coordinate Translator) — 겹치는 영역에서 좌표를 변환하는 함수

**핵심 인물과 일화:**

🧑‍🔬 **베른하르트 리만** (1826–1866)
1854년, 괴팅겐 대학의 교수자격 강연(Habilitationsvortrag). 리만은 세 개의 주제를 제출했고, 가우스가 가장 어려운 세 번째 — 기하학의 기초에 관하여 — 를 골랐다. 리만은 준비 기간 내내 신경쇠약에 시달렸다고 한다. 그 강연 "Über die Hypothesen, welche der Geometrie zu Grunde liegen"에서 그는 n차원 매니폴드의 개념을 처음으로 제시했다. 가우스는 강연을 듣고 매우 깊은 인상을 받았으며, 집으로 돌아가는 길에 동료에게 경탄을 표했다고 전해진다. 리만은 39세에 결핵으로 사망했고, 이 강연은 사후 1868년에야 출판되었다.

**시각화 아이디어:**
- 🌐 **지구본 → 지도책**: 지구본을 여러 장의 종이로 감싸고, 각 종이가 하나의 차트. 겹치는 부분을 확대하면 좌표 전이함수가 보인다. JSX 인터랙티브: 지구 위의 점을 클릭하면 해당 차트의 좌표가 표시되고, 겹치는 영역에서는 두 좌표가 동시에 보인다.
- 📊 **차원 계단**: 0차원(점) → 1차원(원) → 2차원(구면) → 3차원(?)으로 올라가면서, 각각을 덮는 데 필요한 최소 차트 수를 보여준다.
- 🎮 **비디오 게임 맵 래핑**: 팩맨이 화면 왼쪽 끝에서 나가면 오른쪽에서 나타나는 것 = 토러스의 좌표 전이함수.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 위상수학 | 매니폴드의 위상적 분류, 오일러 수, 호몰로지 |
| 물리학 | 일반상대론의 시공간 = 4차원 유사-리만 매니폴드 |
| 데이터 과학 | 매니폴드 가설: 고차원 데이터는 저차원 매니폴드 위에 놓여 있다 |
| 로보틱스 | 관절 로봇의 배위공간(configuration space)은 매니폴드 |
| 통계학 | 확률분포의 모임 = 통계적 매니폴드 |

### 2장 — "곡면 위에서 속도를 말하려면"

**출발 문제:** 구면 위를 움직이는 개미의 속도벡터는 어디에 사는가? 구면은 2차원인데, 속도벡터를 3차원 공간에 빌붙여서 표현해야 하는가?

**패턴:** 각 점에서 가능한 모든 속도 방향이 이루는 공간이 있다. 이 공간은 매니폴드 자체와는 다른, 각 점에 붙은 "별도의 방"이다.

**정리:** 매니폴드 위의 각 점에는 그 점에서의 모든 방향을 담는 벡터공간이 자연스럽게 정의된다.

**정의:**
- **접선벡터** (순간 속도 / Instantaneous Velocity) — 곡선의 한 점에서의 방향과 크기
- **접선공간** (방향의 방 / Direction Room, TₚM) — 한 점에 붙은 모든 가능한 방향의 집합
- **접선다발** (방향의 방 모음 / Direction Room Bundle, TM) — 모든 점의 접선공간을 합친 것
- **벡터장** (바람 지도 / Wind Map) — 매니폴드 위의 매 점에 벡터를 하나씩 부드럽게 배정한 것
- **여접공간** (측정의 방 / Measurement Room, T*ₚM) — 접선벡터를 입력받아 숫자를 뱉는 선형함수들의 공간
- **1-형식** (측정 지도 / Measurement Map) — 매 점에서 접선벡터를 실수로 보내는 부드러운 함수

**핵심 인물과 일화:**

🧑‍🔬 **샤를 에레스만** (Charles Ehresmann, 1905–1979)
파이버 다발(fiber bundle)의 공리적 정의를 확립한 프랑스 수학자. 에레스만은 원래 위상수학자였는데, 카르탕(Élie Cartan)의 이동표구(moving frame) 이론을 공리화하려는 시도에서 접선다발의 현대적 정의에 도달했다. 그의 부인 앙드레 에레스만도 수학자로, 남편 사후 그의 전집을 편찬했다.

🧑‍🔬 **소피아 코발레프스카야** (Sophia Kovalevskaya, 1850–1891)
직접적으로 접선공간을 정의한 것은 아니지만, 미분방정식의 해가 "곡면 위의 곡선"이라는 기하학적 관점의 선구자. 여성이라는 이유로 러시아 대학에 입학할 수 없어 형식적 결혼을 하고 독일로 건너가 바이어슈트라스에게 사사. 세계 최초의 여성 수학 교수 중 한 명이 되었다.

**시각화 아이디어:**
- 🐜 **구면 위의 개미**: 구면 위 한 점에 개미를 올리고, 그 점에서 가능한 모든 이동 방향을 펼쳐 보여주는 접평면. JSX: 구면 위의 점을 드래그하면 접평면이 따라 움직인다.
- 🌊 **바람장 시각화**: 지구 위의 바람 패턴 = 구면 위의 벡터장. "털 달린 공은 빗을 수 없다" (모공정리) → 바람이 0인 점이 반드시 존재한다는 것을 시각적으로 보여준다.
- 📐 **접선 vs 여접**: 접선벡터(화살표)와 1-형식(등고선)의 쌍대성. 화살표가 등고선을 관통하는 횟수 = 두 양의 자연스러운 페어링.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 미분방정식 | 벡터장의 적분곡선 = 미분방정식의 해 |
| 물리학 | 상태공간, 위상공간(phase space)은 여접다발 T*M |
| 기계학습 | 매개변수 공간에서의 그래디언트 = 접선벡터(정확히는 여접벡터) |
| 유체역학 | 유체의 속도장 = 매니폴드 위의 벡터장 |
| 위상수학 | 모공정리(Hairy Ball Theorem): 짝수 차원 구면의 벡터장은 영점이 반드시 존재 |

### 3장 — "거리를 재는 눈금이 장소마다 다르다"

**출발 문제:** 메르카토르 도법의 세계지도에서 그린란드는 아프리카와 비슷한 크기로 보인다. 실제로는 아프리카가 14배 넓다. 왜 이런 일이 벌어지는가?

**패턴:** 지도에 그려진 1cm가 실제 거리에서 의미하는 바가 장소마다 다르다. 각 점마다 "이 근처에서 거리를 이렇게 재라"는 규칙이 있어야 한다.

**정리:** 매니폴드 위에서 곡선의 길이, 각도, 넓이를 정의하려면 각 점의 접선공간에 내적을 부여해야 한다. 이 내적이 점마다 부드럽게 변하면 곡선의 길이를 적분으로 정의할 수 있다.

**정의:**
- **계량 텐서** (국소 눈금 / Local Ruler, gᵢⱼ) — 각 점에서 접선벡터 두 개를 받아 "내적"을 돌려주는 규칙
- **리만 계량** (부드러운 자 / Smooth Ruler) — 점마다 부드럽게 변하는 양의 정부호 계량 텐서
- **선소** (미소 거리 / Infinitesimal Distance, ds²) — ds² = gᵢⱼ dxⁱ dxʲ, 무한소 수준에서의 거리 공식
- **등거리 사상** (자를 보존하는 변환 / Ruler-Preserving Map) — 계량을 바꾸지 않는 매니폴드 사이의 함수

**핵심 인물과 일화:**

🧑‍🔬 **카를 프리드리히 가우스** (1777–1855)
1818년부터 1832년까지, 가우스는 하노버 왕국의 측지 측량을 지휘했다. 산꼭대기에 올라가 거울과 망원경으로 삼각측량을 하는 고된 현장 작업이었다. 이 실무 경험이 곡면론의 대작 *Disquisitiones generales circa superficies curvas* (1827)의 토양이 되었다. 이론이 실무에서 태어난 가장 극적인 사례 중 하나. 전설에 따르면 가우스는 세 산꼭대기의 삼각형 내각의 합을 측정하여 공간이 실제로 휘어져 있는지 검증하려 했다고도 하지만, 이 이야기의 진위는 논쟁적이다.

🧑‍🔬 **게라르두스 메르카토르** (1512–1594)
1569년 세계지도를 만든 플랑드르의 지도 제작자. 항해에서 등각 항로(rhumb line)가 직선으로 보이도록 설계한 도법. 이 도법은 각도를 보존하지만(등각사상, conformal map), 면적을 심하게 왜곡한다. "각도 보존 vs 면적 보존"이라는 트레이드오프는 리만 계량의 본질을 드러낸다.

**시각화 아이디어:**
- 🗺️ **왜곡 격자**: 지구본 위의 정사각형 격자가 메르카토르 도법으로 펼쳐지면 어떻게 변형되는지. JSX: 도법을 전환(메르카토르 ↔ 정적도법 ↔ 람베르트)하면 격자가 실시간으로 변한다. 각 도법이 보존하는 것과 왜곡하는 것을 색으로 표시.
- 📏 **늘어나는 자**: 곡면 위에 자를 올려놓으면, 자의 눈금이 곡면의 계량에 따라 늘어나거나 줄어든다. 포앙카레 원판 위에서 원점 근처의 1cm와 경계 근처의 1cm가 실제로는 전혀 다른 거리.
- 🎵 **음악적 비유**: 계량 = 박자와 템포. 같은 멜로디(곡선)도 템포(계량)가 다르면 길이(연주 시간)가 달라진다.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 일반상대론 | 민코프스키 계량(양의 정부호가 아닌 부정부호): 시간과 공간의 부호가 다르다 |
| 정보기하학 | 피셔 정보행렬 = 확률분포 공간 위의 리만 계량 |
| 탄성역학 | 변형 텐서 = 변형 전후의 계량 차이 |
| 컴퓨터 그래픽 | 텍스처 매핑 = 매니폴드 사이의 사상, 왜곡 최소화 |
| 최적수송 | 바서슈타인 거리: 확률분포 공간의 또 다른 계량 |

### 4장 — "미분했더니 좌표가 섞여 들어온다"

**출발 문제:** 평면에서 벡터장을 미분하는 것은 자연스럽다 — 각 성분을 편미분하면 된다. 그런데 구면 위에서 같은 일을 하면, 원래 벡터장과 무관한 엉뚱한 항이 튀어나온다. 좌표계가 휘어져 있기 때문이다. 어떻게 하면 "좌표의 잡음"을 걸러내고 순수한 변화량만 볼 수 있는가?

**패턴:** 편미분의 결과에서 좌표계의 변화를 보정하는 항을 빼면, 좌표계에 의존하지 않는 미분이 된다. 이 보정 항 자체가 "공간이 어떻게 휘어져 있는가"의 정보를 담고 있다.

**정리:** 매니폴드 위에서 텐서를 좌표 독립적으로 미분하려면 추가적인 구조 — 접속 — 이 필요하다. 계량이 주어지면 자연스러운 접속(레비-치비타 접속)이 유일하게 결정된다.

**정의:**
- **접속** (방향 비교기 / Direction Comparator, ∇) — 서로 다른 점의 접선벡터를 비교하는 규칙
- **공변미분** (보정된 미분 / Corrected Derivative, ∇ᵥW) — 벡터장 W를 방향 v로 미분하되 좌표 잡음을 보정한 것
- **크리스토펠 기호** (좌표 보정값 / Coordinate Correction Term, Γⁱⱼₖ) — 공변미분에서 편미분에 더해지는 보정 항
- **레비-치비타 접속** (계량이 정하는 유일한 비교기 / Metric-Compatible Torsion-Free Comparator) — 계량을 보존하고 비틀림이 없는 유일한 접속

**핵심 인물과 일화:**

🧑‍🔬 **엘빈 브루노 크리스토펠** (1829–1900)
1869년, 크리스토펠은 "이차 미분형식의 변환에 관하여"라는 논문에서 Γⁱⱼₖ를 도입했다. 그의 관심사는 순수하게 대수적이었다 — 이차형식이 좌표변환에서 어떻게 바뀌는지를 추적하는 기계적인 장치. 기하학적 의미(평행이동, 접속)는 50년 후 레비-치비타와 카르탕에 의해서야 밝혀졌다. 이름이 기호에만 남고 기하학적 영광은 후대가 가져간 비운의 인물.

🧑‍🔬 **툴리오 레비-치비타** (1873–1941)
이탈리아 파도바 출신의 수학자. 리치 텐서 해석(tensor calculus)의 공동 개발자. 1917년, 아인슈타인의 일반상대론에 영감을 받아 "평행이동"이라는 기하학적 개념을 도입했다. 이로써 크리스토펠 기호가 순수 계산 장치에서 심원한 기하학적 의미를 갖게 되었다. 유대인이었던 그는 1938년 이탈리아의 인종법으로 교수직을 잃고, 1941년 로마에서 세상을 떠났다.

**시각화 아이디어:**
- 📐 **극좌표의 함정**: ℝ² 위의 벡터장 (1, 0)을 극좌표로 바꾸면 편미분이 0이 아니다. 직교좌표 vs 극좌표에서 같은 벡터장의 편미분을 나란히 보여주고, 차이가 크리스토펠 기호임을 시각화.
- 🔧 **보정 항 슬라이더**: JSX에서 편미분 결과 + Γ 보정 항을 분리하여 보여주고, 보정 항을 on/off하면 결과가 어떻게 달라지는지 보여준다.
- 🧭 **GPS 보정**: GPS 내비게이션이 지구 곡률을 보정하는 과정의 비유 — 평면 위의 직진과 구면 위의 직진이 다르듯, 미분도 보정이 필요하다.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 게이지 이론 | 양-밀스 접속 = 파이버 다발 위의 접속, 크리스토펠 기호 → 게이지 퍼텐셜 |
| 신경망 | 자연 경사법의 핵심: 매개변수 공간의 접속이 학습 역학을 결정 |
| 수치해석 | 매니폴드 위의 ODE 풀이: 보정 없이 적분하면 해가 매니폴드를 이탈 |
| 일반상대론 | 크리스토펠 기호 = 중력장의 좌표 표현 |
| 정보기하학 | α-접속족: 계량이 정하는 접속 외에 무한히 많은 접속이 존재 |

### 5장 — "벡터를 옮기면 달라진다"

**출발 문제:** 북극에 서서 남쪽을 가리키는 화살표를 들고 있다. 이 화살표를 — 절대 회전시키지 않고 — 적도까지 가져간 뒤, 적도를 따라 90° 이동하고, 다시 북극으로 돌아온다. 화살표는 원래 방향을 가리키고 있을까?

**패턴:** 곡면 위에서 벡터를 "회전시키지 않고" 옮기는(평행이동하는) 것은 경로에 의존한다. 같은 출발점, 같은 도착점이라도 다른 경로로 옮기면 결과가 다르다. 이 경로 의존성이야말로 "공간이 휘어져 있다"는 것의 본질이다.

**정리:** 닫힌 경로를 따라 평행이동한 벡터의 회전량은 그 경로가 둘러싼 영역의 곡률의 적분과 같다 (홀로노미).

**정의:**
- **평행이동** (방향 택배 / Direction Shipping) — 접속을 사용해 벡터를 경로를 따라 "회전 없이" 옮기는 것
- **홀로노미** (되돌아옴의 어긋남 / Return Discrepancy) — 닫힌 경로를 따라 평행이동한 벡터의 최종 회전량
- **홀로노미 군** (어긋남의 모임 / Group of Return Discrepancies) — 한 점에서 가능한 모든 홀로노미의 집합

**핵심 인물과 일화:**

🧑‍🔬 **장 밥티스트 조제프 푸코** (Léon Foucault, 1851 실험)
푸코는 회전하는 지구를 증명하기 위해 파리 판테온에 거대한 진자를 매달았다. 진자의 진동면이 천천히 회전하는 이유는 — 물리적 힘이 작용해서가 아니라 — 지구 표면 위에서 진동 방향을 평행이동하면 홀로노미가 생기기 때문이다. 위도 φ에서 하루에 2π sin φ만큼 회전. 평행이동의 가장 극적인 물리적 시연.

🧑‍🔬 **마이클 베리** (Michael Berry, 1984)
양자역학에서 "베리 위상"을 발견. 양자 상태를 매개변수 공간의 닫힌 경로를 따라 단열적으로 변화시키면, 상태가 추가적인 위상(phase)을 획득한다. 이것은 매개변수 공간 위의 접속에 의한 홀로노미에 다름 아니다. 물리학에서 기하학적 위상이 실험적으로 관측 가능하다는 놀라운 발견.

**시각화 아이디어:**
- 🌍 **구면 삼각형 평행이동**: 구면 위에서 삼각형 경로를 따라 화살표를 옮기는 애니메이션. 경로를 드래그하면 홀로노미(회전각)가 실시간으로 바뀐다. 삼각형의 넓이와 회전각의 관계를 숫자로 표시.
- 🎪 **원뿔 전개도**: 원뿔을 펼치면 평면. 꼭짓점 주위를 한 바퀴 돌 때의 홀로노미 = 결손각(deficit angle). 결손각 슬라이더로 원뿔의 뾰족한 정도를 조절.
- 🔄 **푸코 진자 시뮬레이션**: 위도를 바꾸면 진자의 진동면 회전 속도가 달라지는 것을 시각화.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 양자역학 | 베리 위상 = 매개변수 공간 위의 홀로노미, 아하로노프-봄 효과 |
| 로보틱스 | 비홀로노믹 제약조건: 평행주차가 가능한 이유 (측방 이동 불가하지만 홀로노미로 달성) |
| 제어이론 | 비가환적 제어 입력의 효과가 경로 의존적 |
| 게이지 이론 | 윌슨 루프 = 게이지 접속의 홀로노미 |
| 기계학습 | 매개변수 공간에서의 경로 의존적 학습: 같은 시작-끝 모델이라도 학습 경로가 다르면 일반화 성능이 다를 수 있다 |

### 6장 — "곡면 위의 직선은 뭔가"

**출발 문제:** 서울에서 뉴욕까지 가장 짧은 비행경로는? 세계지도(메르카토르 도법)에서 직선으로 그으면 실제로는 최단경로가 아니다. 곡면 위에서 "직선"이란 무엇인가?

**패턴:** 유클리드 공간에서 직선의 두 가지 특성: ① 최단경로, ② 가속도가 0 (= 접선벡터를 자신을 따라 평행이동하면 변하지 않음). 평면에서는 이 둘이 같지만, 일반 곡면에서는 구분해야 한다.

**정리:** 리만 매니폴드 위에서 (레비-치비타 접속에 대해) 가속도가 0인 곡선과 국소적 최단경로는 일치한다. 이것이 측지선이다.

**정의:**
- **측지선** (관성의 길 / Inertial Path) — 접선벡터를 자기 자신을 따라 평행이동해도 변하지 않는 곡선 (∇ᵧ'γ' = 0)
- **측지선 방정식** (관성의 방정식) — γ̈ᵏ + Γᵏᵢⱼ γ̇ⁱ γ̇ʲ = 0
- **지수 사상** (방향 발사기 / Direction Launcher, expₚ) — 접선공간의 벡터 v를 받아 p에서 v 방향으로 측지선을 단위 시간만큼 따라간 점을 돌려주는 함수

**핵심 인물과 일화:**

🧑‍🔬 **레온하르트 오일러** (1707–1783)
1728년, 오일러는 곡면 위의 최단선(geodesic)을 변분법으로 구하는 문제를 풀었다. 이것은 이전 세대의 베르누이 형제가 제기한 최속강하선(brachistochrone) 문제의 자연스러운 확장이었다. "곡면 위에서 최단경로를 구하라"는 문제가 변분법 → 측지선 방정식 → 미분기하학의 핵심 개념으로 이어지는 흐름.

🧑‍🔬 **헤르만 민코프스키** (1864–1909)
아인슈타인의 대학 시절 스승. 1908년 "이제부터 공간 자체와 시간 자체는 그림자로 사라지고, 오직 둘의 결합만이 독립적 실재를 유지할 것이다"라는 유명한 선언과 함께, 특수상대론을 4차원 시공간의 기하학으로 재구성. 시공간에서의 측지선 = 자유입자의 세계선(world line). 아인슈타인은 처음에 민코프스키의 기하학적 재구성을 "불필요한 학문적 치장"이라고 평했지만, 일반상대론을 만들면서 이 기하학 없이는 불가능하다는 것을 깨달았다.

**시각화 아이디어:**
- ✈️ **대원 항로 시각화**: 3D 지구본에서 두 도시를 클릭하면 대원(great circle)을 그리고, 동시에 메르카토르 도법 위에서 같은 경로가 어떻게 휘어져 보이는지를 나란히 보여준다.
- 🎳 **다양한 곡면 위의 측지선**: 구, 안장, 토러스, 원뿔 위에서 측지선이 어떻게 생겼는지를 비교. JSX: 곡면을 선택하고 시작점/방향을 정하면 측지선이 그려진다.
- 💡 **지수 사상 시각화**: 한 점에서 접선평면(평평한 세계)과 곡면(휘어진 세계) 위에 동시에 직선/측지선을 그려서 차이를 보여준다. 가까이서는 거의 같지만 멀어지면 벌어진다.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 일반상대론 | 자유낙하 = 시공간의 측지선, 중력은 힘이 아니라 시공간의 휘어짐 |
| 최적화 | 리만 최적화: 매니폴드 위에서의 경사하강법 → 접선공간에서 그래디언트를 계산하고, 지수 사상으로 매니폴드 위에 복귀 |
| 정보기하학 | m-측지선 vs e-측지선: 같은 통계적 매니폴드 위에서 접속이 다르면 "직선"이 다르다 |
| 컴퓨터 비전 | 물체 인식에서의 측지거리: 형상 공간에서의 최단경로 |
| 변분법 | 오일러-라그랑주 방정식의 기하학적 해석 |

### 7장 — "얼마나 휘었는지를 어떻게 숫자로 말하나"

**출발 문제:** "이 공간은 휘어져 있다"라고 말은 하는데, 얼마나? 어떤 방향으로? 한 숫자로 말할 수 있는가, 아니면 더 복잡한 구조가 필요한가?

**패턴:** 5장에서 발견한 홀로노미 — 닫힌 경로를 따른 평행이동의 어긋남 — 를 무한소 수준으로 줄이면, "이 점에서 이 평면 방향으로 얼마나 휘어져 있는가"를 말할 수 있다. 이것은 스칼라가 아니라 텐서이다 — 방향에 따라 값이 다르기 때문이다.

**정리:** 공변미분의 교환불가능성(∇ᵤ∇ᵥW - ∇ᵥ∇ᵤW ≠ 0)을 측정하는 텐서가 존재하며, 이것이 곡률의 완전한 정보를 담고 있다.

**정의:**
- **리만 곡률 텐서** (방향별 휘어짐 / Directional Curvature, Rⁱⱼₖₗ) — 두 방향으로 공변미분한 순서를 바꿨을 때의 차이
- **단면곡률** (평면 조각의 휘어짐 / Slice Curvature) — 접선공간의 2차원 평면 하나를 골라 그 방향의 곡률만 본 것
- **비앙키 항등식** (곡률의 대칭율 / Curvature Symmetry Law) — 곡률 텐서의 성분들 사이에 성립하는 항등식들

**핵심 인물과 일화:**

🧑‍🔬 **베른하르트 리만** (재등장)
리만은 가우스의 곡면 곡률을 n차원으로 일반화했다. 그러나 그의 1854년 강연에서는 현대적 표기법(텐서)이 없었기 때문에 모든 것을 산문으로 서술해야 했다. 4차원에서 곡률의 독립 성분이 20개라는 것을 좌표 없이 논증한 것은 경이로운 지적 성취였다.

🧑‍🔬 **그레고리오 리치-쿠르바스트로** (1853–1925)
텐서 해석학(tensor calculus)의 창시자. 이 형식주의가 없었다면 리만의 아이디어는 계산 불가능한 상태로 남아 있었을 것이다. 리치는 제자 레비-치비타와 함께 1900년 장대한 논문을 발표. 아인슈타인은 이 형식주의를 배우는 데 수년이 걸렸고, 그의 친구 그로스만(Marcel Grossmann)에게 도움을 구했다.

**시각화 아이디어:**
- 🔍 **미소 평행사변형**: 곡면 위에서 아주 작은 평행사변형을 따라 벡터를 평행이동. 그 회전량이 (평행사변형 넓이) × (곡률). JSX: 평행사변형의 크기와 방향을 조절하면 회전량이 실시간으로 변한다.
- 🎨 **곡률 히트맵**: 토러스 같은 곡면 위에서 곡률의 값을 색으로 표시. 양의 곡률(구처럼 볼록) = 빨강, 음의 곡률(안장처럼) = 파랑, 0 = 초록.
- 📦 **미분 순서 교환**: ∂x∂y vs ∂y∂x는 같지만(편미분의 교환), ∇ₓ∇ᵧ vs ∇ᵧ∇ₓ는 다르다. 이 차이를 두 경로의 결과 비교로 시각화.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 일반상대론 | 아인슈타인 방정식: Rμν - ½gμνR = 8πTμν (곡률 = 에너지-운동량) |
| 위상수학 | 가우스-보네 정리: 곡률의 적분 = 위상적 불변량 (오일러 수) |
| 기계학습 | 손실 풍경의 곡률 → 학습률 선택, 안장점 탈출 전략 |
| 비교기하학 | 단면곡률의 부호가 매니폴드의 전체 형태를 제약 |
| 재료과학 | 결정의 결함(전위, dislocation) = 곡률과 비틀림의 물리적 실현 |

### 8장 — "안에서만 보고도 휘어짐을 안다"

**출발 문제:** 종이를 원통으로 말면 "휘어졌다"고 느끼지만, 종이 위의 개미에게는 여전히 평평하다 — 삼각형의 내각의 합이 180°이고, 평행선이 만나지 않으니까. 반면 지구본 위의 개미는 삼각형의 내각의 합이 180°를 넘는다는 것을 측정할 수 있다. 이 차이는 무엇인가?

**패턴:** 곡면의 휘어짐에는 두 종류가 있다 — 밖에서 볼 때의 휘어짐(외재적)과 안에서 측정할 수 있는 휘어짐(내재적). 원통은 외재적으로는 휘어져 있지만 내재적으로는 평평하다. 가우스 곡률은 내재적이다.

**정리 (Theorema Egregium, 놀라운 정리):** 가우스 곡률 K는 계량 텐서 gᵢⱼ와 그 1, 2차 편미분만으로 계산된다. 곡면이 어떤 공간에 묻혀 있는지(embedding)와 무관하다.

**정의:**
- **내재적 성질** (안에서 아는 것 / Measurable From Within) — 계량만으로 결정되는 기하학적 양
- **외재적 성질** (밖에서 아는 것 / Visible From Outside) — 임베딩(곡면이 놓인 주변 공간)에 의존하는 양
- **가우스 곡률** (두 주곡률의 곱 / Product of Principal Curvatures, K = κ₁κ₂) — 곡면의 각 점에서의 내재적 휘어짐
- **주곡률** (최대·최소 휘어짐 / Maximum and Minimum Bending, κ₁, κ₂) — 한 점에서 모든 방향의 법곡률 중 극값

**핵심 인물과 일화:**

🧑‍🔬 **카를 프리드리히 가우스** (재등장)
가우스는 이 정리를 "놀라운 정리(Theorema Egregium)"라고 스스로 이름 붙였다 — 겸손한 그가 자신의 결과에 이런 이름을 붙인 것은 극히 이례적이다. 그가 놀란 이유: 가우스 곡률 K = κ₁κ₂는 정의상 주곡률의 곱이고, 주곡률은 곡면의 법선 방향(외재적 개념)으로 정의되는데, 그 곱은 순수하게 내재적이라니! 정의는 외재적인데 결과는 내재적인 것 — 이 기적적인 상쇄가 가우스를 놀라게 한 것이다.

🧑‍🔬 **피에르오시앙 보네** (Pierre Ossian Bonnet, 1819–1892)
가우스-보네 정리의 "보네" 쪽. 가우스 곡률을 전체 곡면에 걸쳐 적분하면 위상적 불변량(오일러 수)이 나온다는 정리. 국소적 기하학(곡률)과 전체적 위상(구멍의 수)이 연결되는, 수학에서 가장 아름다운 정리 중 하나.

**시각화 아이디어:**
- 🍕 **피자 정리**: 피자 조각을 구부리면 늘어지지 않는다 — 한 방향으로 구부리면 수직 방향이 뻣뻣해진다. K = κ₁κ₂이고 원래 K = 0(평면)이었으므로, 한 방향으로 구부리면(κ₁ ≠ 0) 다른 방향이 0이어야 한다. 이것이 피자를 접어 먹으면 늘어지지 않는 이유!
- 📜 **종이접기 vs 풍선**: 종이(K=0, 펼칠 수 있음) vs 풍선(K>0, 찢어야만 펼칠 수 있음)의 차이. JSX: K=0인 곡면(원통, 원뿔)과 K≠0인 곡면(구, 안장)을 전개도로 펼치려 시도하는 인터랙티브.
- 🍩 **토러스의 곡률 지도**: 토러스의 바깥쪽은 K>0, 안쪽은 K<0, 위아래는 K=0. 이들의 적분이 정확히 0 (오일러 수 = 0). 색으로 칠해서 보여준다.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 위상수학 | 가우스-보네 정리: ∫∫ K dA = 2π χ(M), 국소 → 전체 |
| 건축/공학 | 셸 구조물 설계: K>0 (돔), K<0 (쌍곡 포물면, 안장 지붕), K=0 (판) |
| 생물학 | 세포막의 곡률이 단백질 배치를 결정한다. 가우스 곡률이 0이 아닌 곳에서만 특정 단백질이 결합 |
| 지도학 | 가우스 곡률이 0이 아닌 곡면은 등거리적으로 평면에 펼칠 수 없다 → 모든 세계지도는 왜곡을 포함 |
| 정보기하학 | 통계적 매니폴드의 곡률 → Fisher 정보의 "복잡도" |

### 9장 — "곡률을 요약하는 법"

**출발 문제:** 리만 곡률 텐서는 n차원에서 n²(n²-1)/12개의 독립 성분을 가진다. 4차원이면 20개. 이 모든 정보를 항상 다 들고 다녀야 하는가? 물리학이나 기계학습에서 실제로 쓰이는 것은 이 거대한 텐서의 "요약"인 경우가 많다.

**패턴:** 텐서의 성분들을 체계적으로 합산(축약, contraction)하면 더 작은 텐서를 얻을 수 있다. 곡률 텐서를 한 번 축약하면 리치 텐서, 두 번 축약하면 스칼라 곡률.

**정리:** 리치 곡률은 "부피가 어떻게 변하는가"를 측정한다 — 측지선 다발이 수렴하는지 발산하는지. 양의 리치 곡률이면 측지선들이 모이고(구처럼), 음이면 흩어진다(안장처럼).

**정의:**
- **리치 텐서** (부피 변화율 / Volume Growth Rate, Rᵢⱼ) — 곡률 텐서를 한 번 축약한 것
- **스칼라 곡률** (전체 휘어짐의 한 숫자 요약 / One-Number Curvature Summary, R) — 리치 텐서를 다시 축약한 것
- **아인슈타인 텐서** (곡률 - ½ 스칼라 × 계량 / Divergence-Free Curvature Summary, Gᵢⱼ) — 발산이 0인 곡률 요약
- **바일 텐서** (조석력 / Tidal Force Tensor, Cᵢⱼₖₗ) — 리치 텐서로 설명되지 않는 곡률의 나머지 부분

**핵심 인물과 일화:**

🧑‍🔬 **알베르트 아인슈타인** (1879–1955)
1912–1915년, 아인슈타인은 중력을 시공간의 곡률로 기술하는 방정식을 찾기 위해 고투했다. 처음에는 리치 텐서를 사용했지만 (Rμν = Tμν), 에너지-운동량 보존법칙과 양립하지 않았다. 비앙키 항등식을 적용하여 발산이 0인 아인슈타인 텐서 Gμν를 발견한 것이 돌파구. 아인슈타인은 리만 기하학을 배우기 위해 친구 그로스만에게 매달렸고, "자네가 도와주지 않으면 미쳐버리겠네"라고 편지에 썼다.

🧑‍🔬 **리처드 해밀턴** (Richard Hamilton, 1943–2024)
1982년, 리치 흐름(Ricci flow) ∂gᵢⱼ/∂t = -2Rᵢⱼ을 도입. "곡률이 높은 곳은 줄어들고 낮은 곳은 팽창하도록" 계량을 흘려보내는 것. 이것은 열방정식의 기하학적 버전 — 온도 차이가 해소되듯 곡률 차이가 해소된다. 2003년 그리고리 페렐만이 이 방법으로 100년 난제인 푸앵카레 추측을 해결.

**시각화 아이디어:**
- 📦 **측지 공의 부피**: 리만 매니폴드 위에서 한 점을 중심으로 반지름 r인 측지 공의 부피가 유클리드 공의 부피와 어떻게 다른지. 양의 곡률 → 부피가 더 작다, 음의 곡률 → 부피가 더 크다. 슬라이더로 곡률을 바꾸면 공의 부피가 변하는 애니메이션.
- 🌡️ **리치 흐름 시뮬레이션**: 울퉁불퉁한 곡면이 리치 흐름에 의해 점차 균일한 곡률의 곡면으로 진화하는 애니메이션. 열전도 시뮬레이션과 나란히 비교.
- 📊 **곡률 텐서 → 리치 → 스칼라 축약 다이어그램**: n=4에서 20개 성분 → 10개 성분 → 1개 숫자로 정보가 압축되는 과정을 시각적으로.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 일반상대론 | 아인슈타인 방정식, 블랙홀의 특이점, 우주론 |
| 위상수학 | 리치 흐름과 푸앵카레 추측 (페렐만, 2003) |
| 기계학습 | 손실 풍경의 리치 곡률: 매개변수 공간에서 "부피"가 줄어드는 영역 = 학습이 어려운 영역 |
| 최적수송 | 리치 곡률의 하한과 최적수송의 수축 성질 (Sturm, Lott-Villani 이론) |
| 정보기하학 | 통계적 매니폴드의 스칼라 곡률 → 추정의 효율성과 관련 |

### 10장 — "같은 공간에 접속이 두 개?"

**출발 문제:** 정보기하학 교재에서 우리는 "합의 길"(m-접속)과 "곱의 길"(e-접속)을 보았다. 4장에서 배운 레비-치비타 접속은 계량으로부터 유일하게 결정되었는데, 어떻게 같은 매니폴드 위에 다른 접속이 존재할 수 있는가?

**패턴:** 레비-치비타 접속은 "계량을 보존하고 비틀림이 없는" 유일한 접속이다. 이 두 조건 중 하나를 포기하면 — 특히 "계량 호환성"을 쌍대적 의미로 일반화하면 — 접속의 쌍이 자연스럽게 나타난다.

**정리 (아마리-나가오카):** 통계적 매니폴드 위에서, 계량 g와 3차 텐서 C(큐빅 텐서)가 주어지면, 접속 ∇⁽ᵅ⁾의 1-매개변수족이 자연스럽게 정의된다. α와 -α에 대응하는 두 접속은 계량에 대해 쌍대이다.

**정의:**
- **쌍대 접속** (거울 접속 / Mirror Connection, ∇ and ∇*) — g(∇ₓY, Z) + g(Y, ∇*ₓZ) = X·g(Y,Z)를 만족하는 접속 쌍
- **α-접속** (α-기울기 비교기 / α-Tilted Comparator, ∇⁽ᵅ⁾) — α = 0이면 레비-치비타, α = ±1이면 e/m-접속
- **큐빅 텐서** (3차 비대칭 / Third-Order Asymmetry, C) — ∇⁽ᵅ⁾ = ∇⁽⁰⁾ + (α/2)C로, 접속 쌍의 차이를 측정
- **쌍대 평탄** (양쪽 다 곧은 / Dually Flat) — ∇와 ∇*가 모두 곡률 0인 경우. 지수족이 이 조건을 만족

**핵심 인물과 일화:**

🧑‍🔬 **순이치 아마리** (甘利俊一, 1936–)
1982년, 아마리는 통계적 추론의 기하학적 구조를 연구하다가 쌍대 접속의 개념에 도달했다. 그의 동기는 순수하게 통계적이었다: 최대가능도 추정량의 기하학적 성질을 이해하기 위해. "왜 지수족에서 최대가능도 추정이 특별히 좋은가?"라는 질문이 쌍대 평탄성의 발견으로 이어졌다.

🧑‍🔬 **니콜라이 첸초프** (Nikolai Chentsov, 1930–1992)
1972년, 첸초프는 피셔 정보행렬이 "마르코프 사상에 의해 줄어들지 않는 유일한 리만 계량"임을 증명했다. 이것은 피셔 계량의 유일성 정리로, 정보기하학의 기초를 놓았다. 소련의 수학자였기에 서방 학계에 알려지기까지 시간이 걸렸다.

**시각화 아이디어:**
- ↔️ **두 종류의 직선**: 확률 심플렉스 위에서 m-측지선(혼합선, 직선으로 보임)과 e-측지선(지수선, 곡선으로 보임)을 같은 그림에 그린다. 같은 두 점을 잇는 "직선"이 접속에 따라 다르다.
- 🎨 **빛의 혼합 vs 물감의 혼합**: RGB 혼합(합의 길)과 필터 중첩(곱의 길)을 나란히 보여주는 인터랙티브. 색상 A와 B를 선택하면 합의 길과 곱의 길로 연결하는 경로를 보여준다.
- ⚖️ **α-슬라이더**: α를 -1에서 +1까지 움직이면 접속이 연속적으로 변하고, 그에 따라 측지선이 어떻게 변하는지 애니메이션.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 정보기하학 | 전체 이론의 기초: 쌍대 평탄 구조, 자연 매개변수 vs 기대 매개변수 |
| 통계학 | 지수족의 기하학, 충분통계량, 최대가능도 추정의 기하학적 의미 |
| 열역학 | 르장드르 변환: 쌍대 좌표계 사이의 전환 = 에너지 ↔ 엔트로피 |
| 기계학습 | DPO/RLHF에서의 KL 발산 방향성: forward KL vs reverse KL이 다른 이유 |
| 양자정보 | Petz 분류: 양자 상태 공간에서의 단조 계량은 연속적 무한족 |

### 11장 — "거리 아닌 거리, 직선 아닌 직선"

**출발 문제:** KL 발산 D(p‖q) ≠ D(q‖p). 거리인데 비대칭? 삼각부등식도 안 만족한다면 이것은 "거리"라고 부를 수 있는가? 그리고 이 비대칭 "거리"로부터 정의되는 "가장 가까운 점"은 어떤 의미인가?

**패턴:** 발산(divergence)은 거리 함수의 일반화이다. 비대칭이지만 "국소적으로" 리만 계량을 유도한다. 발산의 최소화 = 사영(projection), 쌍대 접속에 대한 측지선이 직교하는 피타고라스 정리가 성립.

**정리 (일반화된 피타고라스 정리):** 쌍대 평탄 공간에서 ∇-측지적 부분매니폴드와 ∇*-측지적 부분매니폴드가 직교하면, 발산에 대해 피타고라스 정리가 성립한다: D(p‖r) = D(p‖q) + D(q‖r).

**정의:**
- **발산** (비대칭 거리 / Asymmetric Distance, D(p‖q)) — 두 점 사이의 "방향 있는 떨어짐"
- **브레그만 발산** (볼록함수가 만드는 비대칭 거리 / Convex Gap Distance) — 볼록함수와 접선의 차이
- **사영** (가장 가까운 점 찾기 / Nearest Point Finder, Π) — 부분매니폴드 위에서 발산을 최소화하는 점
- **m-사영 / e-사영** (합의 사영 / 곱의 사영) — 각각의 접속에 대한 측지선으로 정의되는 사영

**핵심 인물과 일화:**

🧑‍🔬 **솔로몬 쿨백** & **리처드 라이블러** (1951)
쿨백-라이블러 발산을 도입. 원래 동기는 통계적 가설검정에서 두 가설을 구별하는 "정보"의 양을 측정하는 것이었다. 쿨백은 2차 세계대전 중 미 육군의 암호 해독 업무에 종사했으며, 정보이론과 통계학의 교차점에서 일했다.

🧑‍🔬 **레프 브레그만** (Lev Bregman, 1941–)
1967년, 소련의 수학자 브레그만은 볼록 최적화 문제를 풀기 위해 "일반화된 거리"를 도입했다. 이것이 후에 브레그만 발산으로 알려지게 된다. 정보기하학의 맥락에서 이 발산이 쌍대 평탄 구조와 완벽하게 맞아떨어진다는 것은 1980년대에야 밝혀졌다.

**시각화 아이디어:**
- 📐 **정보기하학적 피타고라스 정리**: 확률 심플렉스 위에서 e-사영과 m-사영을 그리고, 발산들 사이의 피타고라스 관계를 숫자로 보여준다. D(p‖r) = D(p‖q) + D(q‖r)이 성립하는 순간을 시각적으로.
- 📊 **KL 발산의 비대칭**: D(p‖q)와 D(q‖p)의 크기를 동시에 보여주는 인터랙티브. p와 q를 드래그하면 두 방향의 발산이 다르게 변한다.
- 🎯 **사영 비교**: 같은 점에서 같은 부분매니폴드로 m-사영과 e-사영을 동시에 그린다. 결과가 다르다는 것 자체가 발산의 비대칭을 반영.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 기계학습 | EM 알고리즘 = e-사영과 m-사영의 교대 반복 |
| 강화학습 | RLHF의 KL 페널티: forward KL vs reverse KL의 선택이 학습 결과를 바꾼다 |
| 변분추론 | 변분 추론 = reverse KL 최소화 = e-사영 |
| 최적화 | 미러 디센트(mirror descent) = 브레그만 사영의 반복 |
| 정보이론 | Rate-distortion 이론에서의 블라후트-아리모토 알고리즘 = 교대 사영 |

### 12장 — "매개변수 공간의 풍경을 걷다"

**출발 문제:** 신경망의 학습은 매개변수 공간에서의 최적화이다. 그런데 보통의 그래디언트 ∂L/∂θ를 따라가면, 매개변수의 좌표계에 의존하는 방향으로 걸어가게 된다. 매개변수를 재매개변수화하면 그래디언트의 방향이 바뀐다. 좌표에 의존하지 않는 "진짜 최급강하 방향"은 무엇인가?

**패턴:** 매개변수 공간이 평평한 유클리드 공간이 아니라 휘어진 매니폴드라면, 최급강하 방향은 계량에 의존한다. 피셔 정보행렬을 계량으로 사용하면 통계적 의미에서 좌표 독립적인 최급강하 방향이 정해진다.

**정리 (아마리, 1998):** 자연 경사(natural gradient) ∇̃L = F⁻¹∇L은 피셔 계량에 대한 최급강하 방향이며, 재매개변수화에 대해 불변이다.

**정의:**
- **자연 경사** (좌표 독립 경사 / Coordinate-Free Gradient, ∇̃L = F⁻¹∇L) — 피셔 계량으로 보정한 그래디언트
- **피셔 정보행렬** (확률적 눈금 / Probabilistic Ruler, F) — 로그가능도의 기울기의 공분산. 통계적 매니폴드의 리만 계량
- **미분동형사상** (모양을 보존하는 변환 / Shape-Preserving Map) — 매니폴드 사이의 부드럽고 역도 부드러운 일대일 대응
- **푸시포워드** (밀어내기 / Push-Forward, f*) — 미분동형사상이 벡터와 텐서를 옮기는 방법

**핵심 인물과 일화:**

🧑‍🔬 **순이치 아마리** (재등장)
1998년, 아마리는 자연 경사법을 제안했다. 핵심 통찰: "보통의 경사법은 유클리드 계량을 가정하지만, 매개변수 공간의 자연스러운 계량은 피셔 정보이다. 이 계량 위에서 최급강하하면 학습이 좌표 선택에 의존하지 않는다." Adam, K-FAC 등의 현대적 최적화 알고리즘은 자연 경사의 다양한 근사로 볼 수 있다.

🧑‍🔬 **조지 다르무아** (Georges Darmois, 1888–1960) & **C. R. 라오** (C. R. Rao, 1920–2023)
피셔 정보행렬을 계량으로 해석한 최초의 인물들. 라오는 1945년 논문에서 이를 명시적으로 기술. 라오는 인도 태생으로, 100세를 넘기며 통계학의 산 역사가 된 인물.

**시각화 아이디어:**
- 🏔️ **풍경 걷기 비교**: 같은 손실 함수 위에서 보통 경사(유클리드)와 자연 경사(피셔)의 궤적을 나란히 보여준다. 자연 경사가 어떻게 매개변수 공간의 왜곡을 보정하는지.
- 🔄 **재매개변수화 불변성**: θ → φ(θ)로 매개변수를 바꾸면 보통 경사의 방향이 달라지지만 자연 경사의 방향은 같다는 것을 시각적으로.
- 🌊 **Normalizing Flow**: 단순한 분포(가우시안)가 복잡한 분포로 변환되는 과정을 밀도의 변형으로 보여주는 애니메이션. 야코비안의 역할을 색상 강도로 시각화.

**이 장 끝에서 연결되는 세계들:**
| 분야 | 연결 |
|------|------|
| 심층학습 최적화 | Adam ≈ 대각 자연 경사, K-FAC ≈ 블록 자연 경사, Muon ≈ 스펙트럴 노름 최급강하 |
| 강화학습 | TRPO/PPO: 정책 공간에서의 자연 경사 + 신뢰 영역 |
| 생성 모델 | Normalizing flow, 확산 모델에서의 스코어 함수 = ∇log p |
| 미분방정식 | Neural ODE: 흐름으로서의 신경망, 미분동형사상으로서의 ResNet |
| 정보기하학 | 이 책의 핵심 연결: 기하학적 언어가 학습 알고리즘의 설계를 인도한다 |
