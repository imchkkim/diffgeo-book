# 에필로그: 언어 너머의 풍경

이 책에서 다루지 못한 지평선:

| 주제 | 왜 중요한가 | 어디로 가는가 |
|------|-------------|--------------|
| **파이버 다발** | 접선다발의 일반화. 게이지 이론의 언어 | 양-밀스 이론, 표준모형 |
| **미분형식과 외대수** | 적분을 좌표 없이 하는 언어. 스토크스 정리의 통합 | 드 람 코호몰로지, 호지 이론 |
| **리 군과 리 대수** | 연속적 대칭의 언어 | 표현론, 입자물리, 로보틱스 |
| **심플렉틱 기하학** | 해밀턴 역학의 무대 | 고전역학, 양자화 |
| **최적수송** | 확률분포를 옮기는 가장 경제적인 방법 | 바서슈타인 GAN, 공정성 |
| **양자 정보기하학** | 밀도행렬 공간의 기하학, Petz 분류 | 양자컴퓨팅, 양자 기계학습 |
